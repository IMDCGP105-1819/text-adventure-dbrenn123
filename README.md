# Start

To start, execute the run file from the project directory:

```
python run.py
```

## Commands

Command syntax is `<action> [<target_object>...]`.

Valid actions include:
- `look [object]`

# Testing

## Requirements:

pytest<br/>
pytest-cov

## Run Tests

Run full suite or specify test file path.

```
pytest [--cov] [path]
```
